# 🎮 IndieScope — Steam インディー市場調査ツール

SteamSpy APIを使って、インディーゲーム開発者が**リリース前の市場調査**をできるWebツールです。

## 機能

- ジャンル・タグを選択するだけで市場データを即取得
- ブルー/イエロー/レッドオーシャン判定
- 販売本数・価格帯・レビュースコアの分布をビジュアル表示
- 売れ筋トップ10タイトルの一覧
- 日本語対応（価格は円換算）

## データソース

[SteamSpy API](https://steamspy.com/api) — 無料・認証不要

## 技術スタック

- HTML / CSS / Vanilla JS（単一ファイル）
- Chart.js
- GitHub Pages でホスティング
